# :eye: إشعار الخصوصية

*آخر تحديث: ٢٠٢٦*

---

مرحبًا بك في حديقة شوق الرقمية. هذا الموقع هو منصة توثيق أكاديمي ومهني شخصي مستضافة باستخدام GitHub Pages ويتم تسليمها عبر Cloudflare. هذا الإشعار يشرح المعلومات التي قد يتم جمعها عند زيارتك وكيفية التعامل معها.

---

## :building_construction: الاستضافة والبنية التحتية

<div class="grid cards" markdown>

-   :cat: **GitHub Pages**

    ---

    هذا الموقع مستضاف على GitHub Pages كموقع ثابت (static site). GitHub قد يعالج بيانات تقنية كجزء من عمليات الاستضافة المعتادة.

-   :cloud: **Cloudflare**

    ---

    المحتوى يتم تسليمه وتأمينه عبر Cloudflare، التي توفر خدمات CDN، DNS، والأمن. Cloudflare قد تعالج بيانات تقنية لأغراض الأداء والحماية.

## :bar_chart: المعلومات التي تُجمع تلقائيًا

عند زيارتك لهذا الموقع، البيانات التقنية التالية قد تتم معالجتها تلقائيًا بواسطة مزودي البنية التحتية لأغراض توصيل المحتوى، المراقبة الأمنية، تحسين الأداء، ومنع الإساءة.

| نوع البيانات | أمثلة |
|---|---|
| **الشبكة** | عنوان IP |
| **المتصفح** | النوع، الإصدار |
| **الجهاز** | نوع الجهاز، نظام التشغيل |
| **التنقل** | الصفحات المحولة منها، عناوين URL المطلوبة |
| **التوقيت** | تاريخ ووقت الوصول |

أنا شخصيًا لا أتتبع الزوار الأفراد أو أبني ملفات تعريف للمستخدمين.

---

## :cookie: ملفات تعريف الارتباط (Cookies)

Cloudflare قد تستخدم ملفات تعريف ارتباط أساسية للحماية من البوتات وموازنة التحميل وتحسين الأداء. هذا الموقع قد يستخدم أيضًا التخزين المحلي أو ملفات تعريف الارتباط لاختيار اللغة (إنجليزي/عربي) وتفضيل المظهر (وضع النهاري/الليلي).

لا يتم نشر أي ملفات تعريف ارتباط للإعلانات أو التتبع السلوكي عمدًا.

## :email: معلومات الاتصال

إذا اخترت التواصل معي مباشرة عبر البريد الإلكتروني أو المنصات المرتبطة، فأنت تقدم طواعية اسمك، بريدك الإلكتروني، وأي معلومات تتضمنها رسالتك. هذه المعلومات تُستخدم فقط للرد عليك ولا تُشارك لأغراض تجارية.

---

## :link: روابط خارجية

هذا الموقع قد يحتوي على روابط لمنصات خارجية مثل GitHub، LinkedIn، أو مواقع أخرى. تلك المنصات تعمل وفق سياسات الخصوصية الخاصة بها.

---

## :no_entry_sign: مشاركة البيانات

أنا لا أبيع، أؤجر، أو أتاجر بالمعلومات الشخصية. أي بيانات تقنية تتم معالجتها تحدث عبر GitHub و Cloudflare كمزودي بنية تحتية فقط.

---

## :lock: أمن البيانات

Cloudflare توفر الحماية وتصفية حركة المرور. GitHub Pages توفر البنية التحتية للاستضافة. بينما يتم اتخاذ إجراءات معقولة لحماية الموقع، لا يمكن لأي نظام عبر الإنترنت ضمان الأمان المطلق.
