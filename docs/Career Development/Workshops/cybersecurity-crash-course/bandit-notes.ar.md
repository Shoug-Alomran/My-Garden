## المستوى 0

**الهدف**
الاتصال بخادم Bandit باستخدام SSH والحصول على كلمة مرور المستوى التالي.

**الأوامر المستخدمة**

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```

**شرح الحل**

* الأمر `ssh` يُنشئ اتصال Secure Shell مع نظام بعيد
* `bandit0@bandit.labs.overthewire.org` يحدد اسم المستخدم والخادم الهدف
* الخيار `-p 2220` يحدد المنفذ 2220 (يُستخدم في Bandit بدل المنفذ الافتراضي 22)
* كلمة مرور المستوى 0 مُعطاة مباشرة: `bandit0`

**كلمة المرور للمستوى التالي**
`bandit0`
