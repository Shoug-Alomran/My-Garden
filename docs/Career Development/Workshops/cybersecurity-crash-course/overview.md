# Cybersecurity Crash Course — Offensive Security

**Host:** Automated Systems and Computing Lab (ASCL), Prince Sultan University  
**Type:** 3-day intensive, in-person workshop  
**Focus:** Offensive security – understanding the attacker’s mindset to design more effective defenses.

This workshop introduced the fundamentals of offensive cybersecurity by walking through the steps an attacker might take against a system or network.

## Objectives

- Gain insight into how attackers think and plan.
- Understand common cyber-attack methodologies.
- Learn how to detect and assess security vulnerabilities.
- Connect offensive techniques with defensive and ethical considerations.
