# إشعار الخصوصية

*آخر تحديث: 2026*

## 1. المقدمة

مرحبًا بكم في **حديقة شوق الرقمية**.

هذا الموقع منصة شخصية لتوثيق الأعمال الأكاديمية والمهنية، ويتم استضافته عبر GitHub Pages وتسليمه وتأمينه من خلال Cloudflare.

يوضح هذا الإشعار نوع المعلومات التي قد يتم جمعها عند زيارتكم للموقع وكيفية التعامل معها.

---

## 2. الاستضافة والبنية التحتية

يتم:

* استضافة الموقع على **GitHub Pages**
* تسليم المحتوى وتأمينه عبر **Cloudflare** (شبكة توصيل المحتوى، وخدمات DNS، والحماية الأمنية)

عند الوصول إلى الموقع، قد تتم معالجة بعض البيانات التقنية من قبل مزودي الخدمة المذكورين كجزء من العمليات الاعتيادية لتشغيل الإنترنت.

---

## 3. المعلومات التي يتم جمعها تلقائيًا

عند زيارة الموقع، قد تتم معالجة البيانات التقنية التالية من قبل مزودي البنية التحتية:

* عنوان بروتوكول الإنترنت (IP)
* نوع المتصفح وإصداره
* نوع الجهاز
* نظام التشغيل
* الصفحات المُحيلِة
* عناوين الصفحات المطلوبة
* تاريخ ووقت الوصول

تُستخدم هذه البيانات من أجل:

* توصيل المحتوى
* المراقبة الأمنية
* تحسين الأداء
* منع إساءة الاستخدام

لا أقوم شخصيًا بتتبع الزوار أو إنشاء ملفات تعريف شخصية لهم.

---

## 4. ملفات تعريف الارتباط (Cookies)

قد تستخدم Cloudflare ملفات تعريف ارتباط أساسية لأغراض:

* الحماية والأمان ومنع الروبوتات
* موازنة التحميل
* تحسين الأداء

كما قد يستخدم الموقع التخزين المحلي أو ملفات تعريف الارتباط من أجل:

* حفظ اختيار اللغة (العربية / الإنجليزية)
* حفظ تفضيل المظهر (الوضع الفاتح / الداكن)

لا يتم استخدام ملفات تعريف ارتباط للإعلانات أو التتبع السلوكي بشكل متعمد.

---

## 5. معلومات التواصل

إذا اخترتم التواصل معي مباشرة عبر البريد الإلكتروني أو من خلال المنصات المرتبطة، فإنكم تقدمون طوعًا:

* الاسم
* عنوان البريد الإلكتروني
* أي معلومات تتضمنها الرسالة

يتم استخدام هذه المعلومات فقط للرد على استفساراتكم، ولا يتم مشاركتها لأغراض تجارية.

---

## 6. الروابط الخارجية

قد يحتوي هذا الموقع على روابط لمنصات خارجية (مثل GitHub أو LinkedIn أو مواقع أخرى).
تخضع تلك المنصات لسياسات الخصوصية الخاصة بها.

---

## 7. مشاركة البيانات

لا يتم بيع أو تأجير أو المتاجرة بالمعلومات الشخصية.

أي معالجة للبيانات التقنية تتم عبر GitHub وCloudflare كمزودي بنية تحتية.

---

## 8. أمن البيانات

توفر Cloudflare خدمات الحماية وتصفية حركة المرور.
وتوفر GitHub Pages خدمات الاستضافة.

وعلى الرغم من اتخاذ تدابير معقولة للحماية، لا يمكن ضمان أمان أي نظام عبر الإنترنت بشكل مطلق.

---

## 9. حقوقكم

إذا تواصلتم معي ورغبتم في حذف معلوماتكم الشخصية، يمكنكم طلب ذلك عبر البريد الإلكتروني.
