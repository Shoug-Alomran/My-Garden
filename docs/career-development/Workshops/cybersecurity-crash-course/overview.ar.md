# دورة الأمن السيبراني المكثفة — الأمن الهجومي

**الجهة المستضيفة:** مختبر الأنظمة الآلية والحوسبة (ASCL)، جامعة الأمير سلطان  
**النوع:** ورشة حضورية مكثفة لمدة 3 أيام  
**التركيز:** الأمن الهجومي — فهم طريقة تفكير المهاجم عشان نقدر نصمّم دفاعات أقوى.

هالورشة قدّمت أساسيات الأمن السيبراني الهجومي من خلال استعراض الخطوات اللي ممكن يتبعها المهاجم عند استهداف نظام أو شبكة.

## الأهداف

- فهم طريقة تفكير المهاجم وكيف يخطّط للهجمات  
- التعرّف على أساليب الهجمات السيبرانية الشائعة  
- تعلّم كيفية اكتشاف وتقييم الثغرات الأمنية  
- الربط بين التقنيات الهجومية والجوانب الدفاعية والأخلاقية
