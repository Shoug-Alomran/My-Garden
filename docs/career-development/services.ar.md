## الخدمات المتاحة

### 1. مواقع البورتفوليو

مناسبة للطلاب وحديثي التخرج الذين يحتاجون إلى:

- عرض واضح للمشاريع
- انطباع أول قوي
- هيكلة محتوى منظمة
- تصميم متوافق مع الجوال

الهدف الأساسي: مساعدة المراجعين وجهات التوظيف على تقييم أعمالك بسرعة.

### 2. مواقع التوثيق الأكاديمي

موجهة لمشاريع التخرج والرسائل الأكاديمية والتسليمات المنظمة.

عادة تتضمن:

- تنقل مبني على الفصول
- أقسام واضحة للمنهجية والنتائج
- تنظيم المراجع والملاحق
- هيكلة جاهزة للتسليم

الهدف الأساسي: رفع وضوح المراجعة ودعم تدفق التقييم الأكاديمي.

### 3. صفحات الورش والدورات

مناسبة للمحاضرين والمنظمين الذين يحتاجون إلى:

- تصميم يبدأ بالأجندة
- قسم موارد مركزي
- مسار إرشادي واضح للمشاركين
- تنظيم واضح للتفاصيل اللوجستية

الهدف الأساسي: تقليل الالتباس ورفع جاهزية المشاركين.

## مبادئ التنفيذ

كل مشروع يُبنى على:

- نطاق واضح وحدود شفافة
- هيكلة نظيفة وسهلة الصيانة
- تنظيم واضح للمحتوى وإدارة المراجعات
- تخطيطات عملية تعمل على الجوال وسطح المكتب

## حدود الخدمة

Blueprint تقدم **خدمات مواقع ثابتة فقط**.  
الخدمات التالية خارج النطاق:

- تطبيقات Full-stack
- أنظمة قواعد بيانات
- منطق المصادقة
- تطوير منتجات SaaS
- تكاملات Backend مخصصة

أي مشروع يحتاج أنظمة ديناميكية يعتبر خارج نطاق الخدمة.

## ملاحظات النطاق والاستضافة

يتم نشر المشاريع عبر استضافة مواقع ثابتة، غالبًا عبر GitHub Pages.

يمكنك الاختيار بين:

- نطاق GitHub Pages مجاني  
  مثال: `username.github.io/project-name`
- نطاق مخصص (يُشترى بشكل منفصل)

إذا لم يتم توفير نطاق مخصص، يبقى المشروع منشورًا على رابط GitHub Pages.

تكاليف شراء النطاق والتجديد والاستضافة الخارجية تقع على العميل ما لم يُتفق على غير ذلك كتابيًا.

## نموذج تواصل سريع

للحصول على تقدير مبدئي بسرعة، أرسل:

- نوع الموقع (بورتفوليو، توثيق أكاديمي، أو ورشة/دورة)
- عدد الصفحات
- الموعد النهائي
- هل تحتاج نطاقًا مخصصًا أم لا
