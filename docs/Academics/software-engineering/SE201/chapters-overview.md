# Chapters Overview

Use this page as the starting point for SE201 chapter notes. The chapters move
from the foundations of software engineering into process models, planning,
requirements, design, construction, and testing.

## Suggested Flow

1. Start with Chapter 1 for practices, ethics, and the role of software engineering.
2. Review Chapters 2-4 to understand process models, agile work, and project management.
3. Study Chapters 5-6 for requirements and design.
4. Finish with Chapters 7-8 for construction and testing.
