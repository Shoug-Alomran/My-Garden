# CYS401 — Fundamentals of Cybersecurity

## Course Topics & Lesson Overview

This course introduces the core concepts, technologies, and practices of cybersecurity in organizational and real-world contexts. Students learn how to understand threats and attacks, apply security principles, and design countermeasures to protect modern systems and information assets.

### Introduction to Cybersecurity

- Cybersecurity concepts and terminology  
- Cybersecurity objectives and goals  
- CIA triad (Confidentiality, Integrity, Availability) and system components  
- Critical information characteristics in cybersecurity  

### Security Foundations and Principles

- Cybersecurity foundations and key elements  
- Threats, attacks, vulnerabilities, and countermeasures  
- Common attack types and their mitigation strategies  
- Threat modeling and STRIDE  
- Protection mechanisms and layered defenses  
- Cybersecurity governance and planning in business  

### Data Security and Information Asset Protection

- Information assets and asset domains  
- Due care vs. due diligence  
- Information classification and classification procedures  
- Governmental vs. commercial classification schemes  
- Data labeling and protection mechanisms  
- Data sanitization techniques  
- Data ownership roles and responsibilities  
- Data protection legislation and regulations (e.g., GDPR, PDPA)  

### Cryptography Fundamentals

- Basic concepts and goals of cryptography  
- Core cryptographic components and processes  
- Symmetric vs. asymmetric cryptography  
- Digital signatures and data integrity  
- Public Key Infrastructure (PKI) and digital certificates  

### Principles of Security Design

- Security architecture and design principles  
- Common architecture flaws and security issues  
- Protection mechanisms for confidentiality, integrity, and availability  
- Bounds, process isolation, and confinement  

### Computer Architecture Security

- Hardware components and protection rings  
- Memory protection mechanisms  
- Firmware and operating system hardening  
- Input/output security and TEMPEST  
- Server–client systems security (server-side and client-side attacks)  

### Systems Architecture Security

- Database systems security  
- Cloud computing and virtualization with security aspects  
- Internet of Things (IoT) security  
- Web-based systems security and OWASP  
- Mobile device security  

### ICS/SCADA System Security for CPS

- ICS/SCADA systems and their security landscape  
- Vulnerabilities, threats, and challenges  
- Security requirements, policies, and governance  
- Compliance considerations  
- ICS/SCADA systems security in smart grids  

### Managing Identity and Authentication

- Access control models  
- Access control attacks  
- Controlling access to assets  
- Identification and authentication mechanisms  
- Implementing identity management  
- Biometric standards and applications  
- Authentication vs. identification
