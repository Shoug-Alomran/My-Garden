# CS340 - مقدمة في أنظمة قواعد البيانات

يعرّف CS340 الطالب بمفاهيم قواعد البيانات، بنية قواعد البيانات، النمذجة
المفاهيمية، التصميم المنطقي، القيود العلائقية، الجبر العلائقي، SQL، التطبيع،
مشاريع قواعد البيانات الجماعية، والموضوعات الحديثة مثل قواعد البيانات الكائنية
وNoSQL وعمليات MongoDB CRUD.

## مخرجات التعلم

بنهاية CS340، يفترض أن يكون الطالب قادرًا على:

1. وصف مفاهيم قواعد البيانات وبنيتها ومبادئ DBMS.
2. تطوير نماذج مفاهيمية باستخدام ER أو EER أو UML.
3. إنشاء تصميم منطقي لقاعدة البيانات من نموذج ER المفاهيمي.
4. تطبيق التبعيات الوظيفية والصيغ الطبيعية مثل 1NF و2NF و3NF وBCNF.
5. إنشاء مخططات علائقية تستخدم المفاتيح وسلامة الكيان وسلامة المرجعية.
6. العمل بفعالية ضمن فريق لتصميم وتنفيذ وعرض مشروع قاعدة بيانات.
