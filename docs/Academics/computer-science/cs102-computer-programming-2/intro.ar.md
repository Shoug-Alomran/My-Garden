# CS102 - Computer Programming II

يطوّر مقرر CS102 مهارات البرمجة الكائنية المتوسطة باستخدام Java. يبني المقرر
على CS101 ويركّز على تصميم الأصناف، والوراثة، وتعدد الأشكال، والواجهات،
ومجموعات Java، والأصناف العامة، والاستدعاء الذاتي، والملفات، والمصفوفات ثنائية
البعد، والعمل على مشروع برمجي كائني.

## مخرجات التعلم

بنهاية مقرر CS102، يفترض أن يكون الطالب قادرًا على:

1. معالجة البيانات باستخدام الملفات والمصفوفات ثنائية البعد.
2. تمييز مفاهيم البرمجة الكائنية باستخدام الأصناف وتسلسل الأصناف لإعادة
   استخدام الكود.
3. حل مشكلات برمجية باستخدام الوراثة، والأصناف المجردة، والتغليف، وتعدد
   الأشكال، والواجهات.
4. تطبيق هياكل Java Collection Framework مثل القوائم، و`ArrayList`، والخرائط،
   والمجموعات، و`LinkedList`، و`HashSet`، و`HashMap`، و`PriorityQueue`.
5. تطوير واستخدام الأصناف والدوال العامة لدعم إعادة استخدام البرمجيات.
6. تطوير برامج recursive لحل مشكلات قابلة للحل بالتكرار.
7. العمل بفعالية ضمن فريق لتصميم مشروع كائني وتطويره وعرضه.

## مسار دراسة مقترح

1. راجع موضوعات CS101، خصوصًا المصفوفات، والدوال، والملفات، وتتبع الكود.
2. ادرس تصميم الأصناف والتغليف قبل الوراثة.
3. تدرب على الوراثة، وتعدد الأشكال، والواجهات، والتحويل بين الأنواع، وفحص
   النوع معًا.
4. تعلّم Java Collection Framework بالمقارنة بين استخدامات كل بنية.
5. ادرس Generics وRecursion بأمثلة صغيرة قبل الانتقال إلى مسائل أكبر.
6. استخدم المشروع لربط تصميم الأصناف، وإعادة الاستخدام، والمجموعات، والعمل
   الجماعي.
