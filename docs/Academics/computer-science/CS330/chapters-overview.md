# Chapters Overview

Use this page as the starting point for CS330 computer organization notes. The
chapters cover the core hardware and architecture topics used throughout the
course.

## Suggested Flow

1. Review the early chapters for the foundations.
2. Use the middle chapters to connect components, instructions, and performance.
3. Finish with the later chapters and formulas.
4. Use the cheat sheets for exam review.
