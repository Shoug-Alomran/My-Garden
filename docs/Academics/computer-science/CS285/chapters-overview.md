# Chapters Overview

Use this page as the starting point for CS285 discrete mathematics notes. The
chapters cover logic, proof, structures, number theory, induction, counting, and
relations.

## Suggested Flow

1. Build the foundation with logic and proofs.
2. Review structures and number theory.
3. Practice induction, recursion, and counting problems.
4. Finish with relations and advanced counting.
