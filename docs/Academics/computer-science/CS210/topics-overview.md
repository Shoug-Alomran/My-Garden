# Topics Overview

Use this page as the starting point for CS210 data structures and algorithms
topics.

## Suggested Flow

1. Start with lists, stacks, and queues.
2. Move to trees, BSTs, AVL trees, and heaps.
3. Review hash tables.
4. Finish with recursion, merge sort, and quick sort.

For each structure, know the operations, time complexity, and the shape changes
caused by insertion or deletion.
