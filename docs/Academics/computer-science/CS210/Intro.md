# CS210 Data Structures and Algorithms

This course focuses on how data is organized, stored, and processed efficiently. It covers fundamental data structures, algorithm analysis, and practical implementation techniques that are essential for software development and system design.

This section contains my structured study material for CS210, including:

* Topic based notes on core data structures and algorithms
* Algorithm analysis explanations
* Multiple cheat sheets for revision and quick reference

The goal of this section is to provide a clear and practical resource for understanding data structures and algorithms and for preparing for exams and programming tasks.

## Course Topics

### Linked Objects and Data Structures

* How data is represented in memory
* Object references and dynamic structures

### Linked Lists

* Single, double, and circular linked lists
* Traversal, insertion, and deletion operations

### Analysis of Algorithms

* Big O notation
* Time and space complexity
* Iterative versus recursive analysis

### Sorting Algorithms

* Selection sort
* Insertion sort
* Merge sort
* Quick sort

### Stacks and Queues

* LIFO and FIFO structures
* Applications in compilers and operating systems

### Binary Trees and Binary Search Trees

* Tree traversal methods including inorder, preorder, and postorder
* Searching, insertion, and deletion operations

### AVL Trees

* Self balancing trees
* Rotations and height maintenance

### Heaps and Priority Queues

* Binary heaps
* Heap operations and practical applications

### Hashing

* Hash functions
* Collision resolution techniques such as chaining and probing

### Graphs

* Directed and undirected graphs
* Graph representations
* Depth first search and breadth first search
* Real world applications

## Recommended Study Path

1. Start with linked objects and basic data structures
2. Study algorithm analysis early to understand performance implications
3. Move on to linear structures such as lists, stacks, and queues
4. Continue with trees, AVL trees, and heaps
5. Study hashing and graphs toward the end
6. Use the cheat sheets for revision and quick recall before exams
