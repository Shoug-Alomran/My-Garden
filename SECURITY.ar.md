## الإبلاغ عن ثغرة أمنية

إذا اكتشفت ثغرة أمنية، نرجو الإبلاغ عنها بطريقة مسؤولة.

### طريقة الإبلاغ

* البريد الإلكتروني: **[security@shoug-tech.com](mailto:security@shoug-tech.com)**

### المعلومات المطلوبة

* وصف واضح للثغرة
* خطوات إعادة تنفيذ المشكلة
* أي لقطات شاشة أو سجلات (Logs) ذات صلة
* تقييم للأثر إن وُجد

### ماذا تتوقع بعد الإبلاغ

* راح يوصلك تأكيد استلام خلال **48 ساعة**
* يتم التحقيق في البلاغ وتزويدك بالتحديثات عند الحاجة
* نثمّن ونقدّر مبدأ الإفصاح المسؤول

يرجى **عدم** فتح بلاغات عامة على GitHub بخصوص الثغرات الأمنية.
