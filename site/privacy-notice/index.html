    Privacy Notice¶

    Last updated: 2026

    Welcome to Shoug's Digital Garden. This website is a personal academic and professional documentation platform hosted using GitHub Pages and delivered through Cloudflare. This notice explains what information may be collected when you visit and how it is handled.

    Hosting & Infrastructure¶

    • GitHub Pages

      This website is hosted on GitHub Pages as a static site. GitHub may process technical data as part of standard hosting operations.

    • Cloudflare

      Content is delivered and secured via Cloudflare, which provides CDN, DNS, and security services. Cloudflare may process technical data for performance and protection.

    Information Collected Automatically¶

    When you visit this site, the following technical data may be automatically processed by infrastructure providers for content delivery, security monitoring, performance optimization, and abuse prevention.

    Data Type Examples
    Network IP address
    Browser Type, version
    Device Device type, operating system
    Navigation Referring pages, requested URLs
    Timing Date and time of access

    I do not personally track individual visitors or build user profiles.

    Cookies¶

    Cloudflare may use essential cookies for security and bot protection, load balancing, and performance optimization. This website may also use local storage or cookies for language selection (English / Arabic) and theme preference (Light / Dark mode).

    No advertising or behavioral tracking cookies are intentionally deployed.

    Contact Information¶

    If you choose to contact me directly via email or linked platforms, you voluntarily provide your name, email address, and any information included in your message. This information is used only to respond to you and is not shared for commercial purposes.

    Third-Party Links¶

    This website may link to external platforms such as GitHub, LinkedIn, or other websites. Those platforms operate under their own privacy policies.

    Data Sharing¶

    I do not sell, rent, or trade personal information. Any technical data processed occurs through GitHub and Cloudflare as infrastructure providers only.

    Data Security¶

    Cloudflare provides security and traffic filtering. GitHub Pages provides hosting infrastructure. While reasonable measures are taken to protect the site, no online system can guarantee absolute security.

    Your Rights¶

    If you have contacted me and would like your message or personal information removed, you may request deletion via email at any time.

    Contact¶

    For privacy-related inquiries, please reach out directly.
